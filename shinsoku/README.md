# 神速 SHINSOKU

擬似3D（アイソメトリック）の2Dマップで戦うハック&スラッシュRPG。剣・盾・鎧を拾い集め、**攻撃速度を極限まで上げて「神速」**に到達し、モンスターの群れを薙ぎ払うのが目的。2015年頃のMMORPGの質感（リッチなライティング、グロー、半透明HUD）を狙ったブラウザ単体ゲーム。

MICRO ARCADE とは独立したスタンドアロン作品。Vanilla JS + Canvas、ESモジュール、フレームワーク不使用。

## 遊び方
- **移動**：WASD / 矢印キー / 画面左下のバーチャルスティック（タッチ）。地面タップで移動。
- **攻撃**：敵をタップ／クリックでターゲット。射程に入ると攻撃速度（atk/s）に応じて自動で斬り続ける。
- **拾う**：ドロップの上を歩くと自動取得。
- **育成**：右上の ▦ ボタンでステータス／装備／スキル割り振り。

## 仕様の要点
- **戦闘式**：与ダメージ = `攻撃力 × ばらつき − 敵防御`。被ダメージ = `敵攻撃力 − 防御力`、**回避**で完全回避（ダメージ0）。
- **攻撃速度スケーリング**：実効 atk/s = `基礎 × (1 + 攻撃速度% / 100)`。**8 atk/s で「神速」モード**に突入し、残像・グロー・粒子が激化。スタックを積めば 60〜100 atk/s に到達可能。
- **HP吸収**：与ダメージの `HP吸収%` を回復。高HPモンスター（オーガ／ゴーレム）を倒すには速度＋吸収のビルドが鍵。
- **ドロップオプション**：剣→攻撃速度%、盾→回避%、鎧→HP吸収%、その他フラット（攻撃力／最大HP／防御）。高ティアほど良ドロップ。
- **レベル**：撃破でXP。レベルアップで各ステータス上昇＋スキルポイント（攻撃力／最大HP／移動速度）。
- **神（特殊）ボタン**：5秒間 攻撃速度+400%＆無敵の「神速発動」。クールダウン30秒。
- セーブは `localStorage`（自動）。タイトルの「セーブを消去」でリセット。

## キャラクター
プレイヤーは**女剣士**（ベクター・関節アニメ・描き込み済み：髪ハイライト、顔、首鎧、ベルト、肩当て、膝当て、籠手）。**剣は右手・盾は左手**で固定。デフォルトは剣を下ろし、攻撃時に振り上げて→振り下ろして斬る。移動方向に応じて**正面・後ろ姿・横向き**の3ビューで歩行アニメが切り替わる。

## 演出・サウンド
- **環境演出**：エリア別の浮遊パーティクル（ダンジョン＝火の粉/塵、草原＝花粉/蛍、森＝落ち葉/胞子）。
- **サウンド**：Web Audio による合成効果音（攻撃/会心/撃破/レベルアップ/スキル/取得/被弾）とエリア別アンビエント。右下の 🔊 でミュート切替（設定は保存）。

## スキル（アクティブ・時限バフ）
下部スロットの3スキルをタップで発動。レベルが上がると効果と持続時間が伸びる。レベルアップ時にスキルポイントを ▦ パネルで割り振る。
- **神速**：攻撃速度 +85%/Lv（神速モードのトリガー）
- **剛力**：攻撃力 +30%/Lv
- **活力**：最大HP +45/Lv（発動時に回復）

## エリア（ダンジョンマップ）
3エリアがループ：**地下牢（ダンジョン）→ 草原 → 森 → 地下牢**。各エリアに出入り口（光るゲート）があり、踏むと次エリアへ。エリアごとに地形（石床＋魔法陣／草原／森）・出現モンスター・**ボス**が異なる。画面上部に**ボスHPバー**、右上に**ミニマップ**。

## 装備ドロップ
モンスターは①**オプション増加アイテム**（恒久ステUP）と②**武器・防具・盾**（複数オプション付き、レア度あり）を落とす。良い装備ほどオプションが多い。ボスは高レア装備を確定ドロップ。

## モンスター（5ティア）
コボルト（群れ）→ リザードマン → レイス（高回避・俊足）→ オーガ（重装）→ ゴーレム（ボス級・高防御）。サイズはキャラ同等〜大きめ。徘徊（歩行アニメ）する。

**アグロ仕様**：基本は**パッシブ＝寄ってこない**（攻撃すると反撃して戦闘開始）。**オーガのみ縄張り型**で、約5マスに近づくと感知して向かってくる。

## 舞台（ダンジョンアリーナ）
Lineage 2 系の薄暗いダンジョンをイメージ。薄暗い石床、中央に**光る巨大魔法陣**（回転するルーン環＋神速ピンクの核）、周囲を**石柱＋揺れるトーチ（暖色光）と石壁リング**が囲む円形アリーナ。強いビネットでゴシックな陰影。プレイ範囲はアリーナ内（半径約11マス）にクランプされ、スポーン・移動もその中。

## ローカル実行
```
node shinsoku/server.js   # http://localhost:4180
```

## 構成
- `index.html` / `css/style.css` — シェルとHUD。
- `js/main.js` — 起動（タイトル→ゲーム）。
- `js/game.js` — エンジン：アイソメ投影・カメラ・戦闘・レベル・神速・入力・HUD・セーブ。
- `js/sprites.js` — キャラ／モンスター／ドロップ／エフェクトの描画。
- `js/content.js` — モンスター・アイテムオプション・ドロップ／スポーンテーブル（純データ）。
