# AbyssGenesis（仮称）設計計画書・仕様書

> 神の視点で生命環境を見守り、介入によって生命の繁栄・進化・絶滅を導く、ライフゲーム的シミュレーション。
> 本書はパイロット（水中・カンブリア多様化ステージ）の設計と、将来構想をまとめたもの。

- バージョン: 設計まとめ v1（プロトタイプ v6 時点）
- 対象プラットフォーム: スマートフォン（縦画面）／PCブラウザでも動作
- ジャンル: 神視点 × エージェント型生命シミュレーション × 進化・収集

---

## 1. コンセプト

プレイヤーは「神」として、生命の環境に介入する。生命そのものは直接操作せず、**環境を揺らす**ことで、生命が自律的に繁栄・進化・絶滅していく様を俯瞰し、また個別に観察する。

- ライフゲーム的な自律シミュレーションが土台
- 介入は「恵み（甘やかす）」と「試練（鍛える）」の二元論
- 地球生命の進化を俯瞰する壮大な系譜を背骨に持つ
- 短時間でターンが進み、見ていて楽しい
- メタ視点（俯瞰）と個別状況（個体観察）の両方を行き来できる

### コアバリュー（差別化の核）

1. **因果推論モデルを進化エンジンに採用** — 曝露・アウトカム・中間因子・修飾因子・交絡因子という疫学的構造を、そのままゲームメカニクスにする。他にない深さと知的好奇心の刺激。
2. **「環境に介入する神」への操作対象の固定** — Spore の失敗（ステージごとに別ゲームになる）を構造的に回避する。
3. **観賞型のテンポ** — 強い一手を打ち、しばらく放置して変化を眺め、流れが弱まったら次の一手。

## 2. 設計思想・重要な判断

ここまでの議論で確定した方針と、棄却した案を記録する。

| 論点 | 決定 | 理由 |
|---|---|---|
| 主目標 | **多様性の収集（図鑑）** を主、**世界の解放** をメタ進行に | 収集は中毒性が高く、進捗が定量化できKPIになる |
| 進化の作る順番 | 面白さのピーク（**多様化期**）からパイロット化。序盤の化学段階は演出に留める | 下から順（タンパク質→細胞）に作ると地味な序盤で離脱する |
| 操作対象 | 常に「環境」。生命は直接操作しない | ステージ間で操作が変わる Spore の罠を回避 |
| 数式の見せ方 | **二層構造**（表＝触感的なバー/演出、裏＝因果ノート） | Easy to learn, hard to master |
| 交絡因子 | ランダムノイズ化は禁止。「**調べれば見える隠れ環境変数**」にする | 理不尽な雑音ではなく「発見の喜び」に変える |
| 修飾因子 | 攻略の核（環境で介入効果が激変） | 「環境を読む」面白さを生む |
| 超レア種 | 架空種・キーストーン種は**棄却**。**既存種のレア変異体**（色違い的）を採用 | 学術的信頼性を壊さずに収集の興奮を出せる |
| 介入の経済 | 神力コスト＋クールダウン。同一介入の連打は**効果飽和** | 連打作業ゲー化を防ぎ、観賞型テンポを成立させる |
| 天啓ゲージ | タップでは溜まらない。**豊かな生態系の維持**で蓄積 | 連打で稼ぐループを断つ |

## 4. ゲームメカニクス

### 4.1 介入

- **恵み（タップ）**: 光を注ぐ。光エネルギーを上げ、生産者を活気づける。
- **試練（長押し／別ボタン）**: 進化圧（変異圧）をかける。淘汰圧で個体にダメージ。
- **神力コスト**: 介入1回につき神力を1消費。神力は最大5、数ターンに1回復。連打不可。
- **効果飽和**: 対象の値が高いほど、同じ介入の効果は逓減（`add = base × 修飾 × (1 − level/100)`）。高い状態で押しても無駄＝「待つ」が最適になる。

### 4.2 数式エンジン（因果推論モデル）

進化圧（新種が生まれる駆動力）を、疫学的な構造で計算する。

```
進化圧 = exp( β·曝露·修飾 + 中間因子 + 交絡 )
```

| 疫学因子 | ゲームでの役割 |
|---|---|
| 曝露因子 | プレイヤーのタップ（恵み／試練） |
| 修飾因子 | 環境による効果の変化（攻略の核） |
| 中間因子 | 光エネルギー・変異圧などの蓄積ゲージ（可視化の主役） |
| アウトカム | 新種の誕生（図鑑登録） |
| 交絡因子 | 隠れ環境変数（例: 鉄分）。調べると可視化 |

#### 環境による修飾係数（例）

| 環境 | 光への修飾 | 捕食圧への修飾 |
|---|---|---|
| 温・明・浅 | ×2.0 | ×1.0 |
| 温・明・深 | ×1.2 | ×1.3 |
| 冷・暗・深 | ×0.3 | ×1.5 |

#### 交絡因子の扱い

「鉄分」が裏で光エネルギーを底上げしている。最初は不可視（「未知の寄与 ?」）。蓄積が進むと「⚠ 未知の寄与を検出」→さらに進むと「🔍 交絡の正体は鉄分」と可視化され、図鑑的な発見報酬となる。

### 4.3 天啓ゲージ（神の必殺）

- タップでは溜まらない。**3種以上が生息する豊かな生態系**を維持すると自然に蓄積。
- 満タンで発動可能。**一斉進化**（光・変異圧を一気に底上げ）。
- 設計鉄則: 「近道」でなく「触媒」。発動しても、狙った種になるかは環境設計次第＝技術介在を残す。
- 将来: 発動を「一斉進化／多様性爆発／環境改変」の3択にして「神の決断」を濃くする。

### 4.4 介入メニュー（将来実装の段階的UI）

通常は2択（恵み／試練）。特定の状況・ステータス・イベントでのみ、介入ボタンをタップすると2〜3のおすすめメニューが開く。

| トリガー | 発動条件 | 選択肢の例 |
|---|---|---|
| 進化の岐路 | 中間因子が閾値の−10以内 | 形態を促す／機能を促す／静観 |
| ウイルス流行 | イベント | 免疫を恵む／あえて流行／静観 |
| 大災害 | イベント | 休眠を促す／適応を促す／静観 |
| 交絡の兆候 | 未知の寄与が蓄積 | 調査する／無視して進める |
| 多様性の停滞 | 一定ターン新種ゼロ | 撹乱を起こす／恵みを集中 |
| 環境遷移 | 環境が境界値に接近 | 環境を維持／変化を許す |

- 競合時の優先順位: イベント系 ＞ 状況系 ＞ ステータス系（必ず1つに絞る）。
- 発動はクールダウンで希少化（乱発すると2択の良さが死ぬ）。

## 5. 生態系シミュレーション（エージェント型）

各個体は実際に泳ぎ、獲物を探し、食べ、殖え、死ぬ。捕食は数式ではなく**画面上の出来事**として起こる。

### 5.1 食物網

```
光 → 原始多細胞・緑藻・葉状藻（生産者・分裂で増殖）
        ↓ 遊泳虫が泳いで捕食 → 死骸を残す
     遊泳虫          三葉虫が死骸を漁る（スカベンジャー）
        ↓
   アノマロカリスが遊泳虫・三葉虫を追って捕食（頂点捕食者）
```

### 5.2 個体パラメータ

- **HP**: 餌が得られれば回復、飢え・放置・試練で減少。0で死亡。
- **齢／寿命**: 寿命を超えると死亡。捕食者は捕食成功で寿命が延びる。
- **状態**: 満腹／空腹／衰弱／高齢。
- 死亡時は**死骸**を残し、ゆっくり沈んでフェード。三葉虫の餌になる。

### 5.3 種ごとの移動パターン・知覚

| 種 | 移動パターン | 知覚範囲 | 食性 |
|---|---|---|---|
| 原始多細胞 | ブラウン運動 | — | 生産者（光） |
| 緑藻・葉状藻 | その場で揺らぐ（固着的） | — | 生産者（光） |
| 遊泳虫 | ジグザグ、中距離で察知し直進捕食 | 中（150） | 原始多細胞・緑藻 |
| 三葉虫 | ゆっくり這う、近距離のみ | 小（80） | 死骸（デトリタス） |
| アノマロカリス | 巡航、遠距離で察知し高速突進（待ち伏せ型） | 大（190） | 遊泳虫・三葉虫 |

### 5.4 個体群ダイナミクス

- 生産者は光が十分なら分裂して増殖（放置で画面が埋まる）。共通の環境収容力で頭打ち。
- 捕食者は満腹かつ高HPで繁殖（HPを消費）。餌がなければ飢えて衰弱死。
- 捕食-被食のブーム＆クラッシュ（振動）が観察できる。
- 原始多細胞は「絶滅しない基盤」として水槽を自走させる（放置でも最低限の生命が残る）。

## 6. 進化・収集システム

### 6.1 進化ツリー（パイロット）

```
                    ┌─ A 光合成系（光の恵み）
原始多細胞 ──────┤
                    ├─ C 運動捕食系（変異圧）★脊椎本流
                    └─（将来）B 濾過系 / D 極限系
★=次ステージへの鍵  ◇=交絡で開く隠し種
```

### 6.2 種（系統）と生物（収集対象）

設計方針: **系統は少数、その中の生物は複数**（少数アーキタイプ × 多数コレクション）。系統＝数式で発火するメカニクス単位、生物＝環境・介入の優勢さで細分される収集対象。

| 系統 | 代表生物（実在の花形を主役に） |
|---|---|
| 葉足・奇蝦類 | アノマロカリス、オパビニア、ハルキゲニア |
| 三葉虫類 | 三葉虫、マルレラ |
| 脊索類 ★ | ピカイア、ハイコウイクチス |
| 有顎魚類 ★ | ダンクルオステウス（魚類ツリー解放→上陸へ） |
| 原始藻類 | グリパニア、緑藻マット |
| 原始植物 ★ | クックソニア（植物ツリー解放） |

- メカニクス分岐（系統）は約12、収集対象（生物）は約25を想定。
- 実装は12〜15種、残りは図鑑に**影シルエット**で表示して収集欲を喚起。

### 6.3 図鑑（学術資料性）

- 和名＋**学名（斜体）**＋分類・時代・体長・食性を記載。
- ゲーム独自に「発見条件」を併記＝攻略ヒント集を兼ねる。
- 影シルエットで未発見種を示し、系統ごとに進捗（例: 4/9）を表示。

### 6.4 レア変異体

- 既存種の低確率変異体（例: **黄金三葉虫** 希少度0.4%）。
- 形態レア（金色・大型）と機能レア（生態系を安定させる等）の2タイプ。
- 学名は実在種を保ち「var.（変異体）」を付すことで図鑑の学術性を保つ。
- 入手難度は通常種をほぼ集めた後半ゲートに置くのが安全（序盤に出すと世界観がブレる）。

## 7. UI/UX

### 7.1 画面構成（メインプレイ）

- 上部: ターン数・環境表示・図鑑進捗・天啓ゲージ
- 中央: **俯瞰ビュー（映像を最大化）**。生命が泳ぎ回る主役エリア
- 下部: 中間因子バー（光エネルギー・変異圧・交絡）／神力ピップ／介入ボタン2つ
- ボトムナビ: 俯瞰／個別／図鑑

### 7.2 カメラ（広角 ↔ 区域ズーム）

- 世界は画面の **3×3＝9区画ぶん** の広さ。
- デフォルトは**広角**で全体のダイナミクスを俯瞰。
- グリッドの区画をタップで**その区域を拡大**観察。「⤢ 広角ビュー」ボタンで戻る（他UIと統一したダークピル型）。
- 将来: 4分割・6分割など分割数の切替も可能。

### 7.3 介入のフィードバック演出

- 恵み: 画面がティールにホワンと発光、全個体が跳ねて淡く光る。
- 試練: 画面が赤く明滅＋シェイク、全個体がひるむ。
- 捕食の瞬間: 捕食者がパクッと膨らむ。誕生時はポップイン。

### 7.4 研究ログ

- 画面占有を避け、アイコン（⚗）からポップアップ表示。
- 最上部に**現在値を代入した進化圧の数式**と生息種数・総個体数を固定表示。
- 介入結果は赤文字トーストで一時表示。

### 7.5 個体観察

- **マウスオーバー（スマホは触れた位置）** で、ポインタ近傍の個体情報を表示。
- 名前・HPバー・齢/寿命・状態をリアルタイム更新。
- 俯瞰の全体把握と個別の詳細を両立。

## 8. 将来設計（ミクロ→マクロの系譜）

地球生命の進化を俯瞰する壮大なアークを背骨に持つ。ただし**全系譜の順番どおりには作らない**。

```
無機物 ─ 細胞 ─ 多細胞 ─【カンブリア多様化】─ 魚類 ─ 上陸 ─ …
                          ▲ ここがパイロット
```

- 序盤の化学段階（タンパク質→細胞）は本編にせず、詩的なオープニング演出に留める。
- 「植物の進化」「魚類の進化」を経て、**地上世界の解放＝生命の上陸**を中期目標に。
- 未実装ステージは鍵マーク／影で「先がある」と匂わせる。
- 水中の次は地上・空・地中・極限世界など、難易度別の世界を順次解放。

## 11. リスクと留意点（経営・開発）

- **スコープ過大**: 全系譜・全世界をフルで作ると資金と時間を溶かす。MVPは水中1枚に絞る。
- **Spore の罠**: 操作対象を「環境」に固定して回避済み。新ステージでも崩さないこと。
- **専門性の翻訳**: 因果モデルは専門家には宝だが一般プレイヤーには統計の試験に見える。表層は触感・演出で、深層に因果を置く二層化を厳守。
- **演出と世界観のバランス**: 架空種を入れない判断を維持。レアは既存種の変異体に限定。
- **収益化**: 深い戦略ゲームはガチャと相性が悪い。買い切り or 広告なしの世界観重視を前提に別途検討。
