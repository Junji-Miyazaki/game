# MICRO ARCADE 企画書

## コンセプト
> 「暇つぶしにやるけどハマる、80年代マイコン／CRTの香りがするミニアーケード集」

グリーン／アンバーCRT風の1画面ゲーム集。起動 → メニューから5本を選択 → 1プレイ30秒〜2分 → 死んだら即リトライ。ハイスコアはブラウザにローカル保存。

memo.txt の3要素「**レトロ・シンプル・中毒性**」と3案（ローグライク／2進数パズル／ハッキング）をベースに、中毒性の高い定番（スネーク・記憶ゲーム）を加えて5本構成にした。

## 3つの設計原則
1. **1画面・スクロールなし** — 論理解像度 360×640 に固定し、端末サイズに合わせて等比拡大（レターボックス）。`touch-action:none` でスクロール／ズームを禁止。
2. **シンプル操作** — スワイプ／タップ／矢印キーのみ。説明不要で触れば分かる。
3. **レトロ演出** — 色数を絞ったパレット（16色未満）、ドット風ビットマップフォント（DotGothic16）、矩形波のBEEP音（WebAudio）、CSSで走査線＋ブラウン管ビネット、`image-rendering:pixelated`。

## 収録ゲーム（5本）
| # | タイトル | 内容 | 操作 | 元ネタ |
|---|----------|------|------|--------|
| 1 | **1-BIT HERO** | 1画面ダンジョン。@で鍵Kを取り出口Dへ、敵Eを避ける。クリアで次レベル | スワイプ／矢印 | memo案2 |
| 2 | **BINARY** | 制限時間内に目標の10進数を6bitのトグルで作る | タップ | memo案1 |
| 3 | **HACK** | 流れるコード群から指定の暗号を時間内にタップ解除 | タップ | memo案3 |
| 4 | **SNAKE** | CRT緑の定番スネーク。食べて伸ばし、壁と自分を避ける | スワイプ／矢印 | 定番中毒枠 |
| 5 | **BEEPER** | 光る順番を記憶して再現するサイモン。矩形波SEを活用 | タップ | 記憶ゲーム |

各ゲーム共通: HUDにSCORE/HI、ゲームオーバーで即リトライ、`back`でメニューへ。メニューからテーマ（GREEN CRT ⇄ AMBER）切替可。

## 技術方針
- **Vanilla JavaScript + Canvas 2D + ESモジュール**。フレームワーク／ビルド不使用。
- 共通エンジン（`js/core/`）が「描画・入力・音・保存・シーン管理」を担い、各ゲームは `Scene` を継承するだけ（疎結合＝1ファイルで1ゲーム）。
- 追加が容易: 新ゲームは `js/games/xxx.js` を作り `main.js` に1行 import するだけ。

## アプリ化への道筋（memo.txt の目標）
今回はブラウザJSでMVPを完成。アプリ化は段階的に:
1. **PWA化（最短）** — manifest＋Service Worker を足せばホーム画面追加・オフライン動作。コードはそのまま。
2. **Capacitor でラップ** — この Web 資産を iOS/Android ネイティブアプリにパッケージし、App Store / Google Play へ。AdMob プラグインで広告（バナー／リワード「復活」／インタースティシャル）も導入可能。
3. **SwiftUI で書き直し（memo推奨ルート）** — 仕様が固まった本命だけを Swift で再実装し、最高の最適化と純正感を得る。

→ まずブラウザ版で「どのゲームが一番ハマるか」をユーザー反応で見極め、勝ち筋に絞ってネイティブ化するのが収益化への近道（memo.txt の MVP 戦略と一致）。

## フォルダ構造
```
MiniRogueGame/
├── index.html          エントリ（canvas + CRTオーバーレイ）
├── server.js           開発用の最小静的サーバ（node server.js）
├── css/
│   └── style.css       CRT演出・1画面フィット・スクロール禁止
├── js/
│   ├── main.js         起動・ゲーム登録・画面遷移の配線
│   ├── menu.js         タイトル＆ゲーム選択メニュー
│   ├── core/           ★共通エンジン
│   │   ├── engine.js   ループ・キャンバス拡大・入力・Scene基底・描画ヘルパ
│   │   ├── input        （engine.js内：スワイプ/タップ/キーボード）
│   │   ├── audio.js     矩形波BEEP（WebAudio）
│   │   ├── storage.js   ハイスコア保存（localStorage）
│   │   └── palette.js   レトロカラーパレット（GREEN/AMBER）
│   └── games/          ★ゲーム本体（1ファイル1ゲーム）
│       ├── rogue.js     1-BIT HERO（実装の手本）
│       ├── binary.js    BINARY
│       ├── hack.js      HACK
│       ├── snake.js     SNAKE
│       └── beeper.js    BEEPER
├── assets/             （将来の画像・フォント・音源用）
├── 企画書.md           本書
├── README.md           動かし方・拡張方法
└── memo.txt            元アイデアメモ
```
